# Low-Code Contract Review Agent

This project demonstrates a low-code Agentic AI approach to contract review.

- No LangChain
- No agent frameworks
- Configuration-driven
- Human-in-the-loop by design

## How to run

1. Install dependencies  
   pip install -r requirements.txt

2. Add your OpenAI API key  
   Update analyze_contract.py

3. Add a contract PDF  
   sample_contract.pdf

4. Run  
   python analyze_contract.py
